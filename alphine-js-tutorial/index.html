Dropwdown

  • 1. x-data="{ isOpen: false }" on the position relative element.
  • 2. x-show="isOpen" on the position absolute element (dropdown).
  • 3. @click="isOpen = !isOpen" on the button element in order to toggle the dropdown.
  • 4. @click.away="isOpen = false" on the dropdown element in order to close the dropdown when you click outside its area.
  • 5. @keydown.escape="isOpen = false" on the button element in order to close the dropdown when you tap escape on button focus.

Modals and x-ref

  • 1. x-data="{ isOpen: false }" on the element that encloses the button and modal.
  • 2. x-show="isOpen" on the position absolute element (modal).
  • 3. @click="isOpen = true; ..." on the open button.
  • 4. @click="isOpen = false" on the close button.
  • 5. x-ref="modalCloseButton" on the button you want to make reference to.
  • 6. "... $nextTick(()=> $refs.modalCloseButton.focus())" on the button the reference callback function should be called.

Tabs

  • 1. x-data="{ tab: 'tab1' }" to set the first tab to active.
  • 2. @click.prevent="tab = 'tab1'" set the first tab active on click.
  • 3. @click.prevent="tab = 'tab2'" set the second tab active on click.
  • 4. @click.prevent="tab = 'tab3'" set the third tab active on click.
  • 5. x-show="tab === 'tab1'" whether to show or hide tab based on active tab.
  • 6. x-show="tab === 'tab2'" whether to show or hide tab based on active tab.
  • 7. x-show="tab === 'tab3'" whether to show or hide tab based on active tab.
  • 8. :class="{ 'bg-white text-blue-700 border-l border-t border-r' : tab === 'tab1' }" conditional rendering label classes based on active tab.
  • 9. :class="{ 'bg-white text-blue-700 border-l border-t border-r' : tab === 'tab2' }" conditional rendering label classes based on active tab.
  • 10. :class="{ 'bg-white text-blue-700 border-l border-t border-r' : tab === 'tab3' }" conditional rendering label classes based on active tab.

Image Selection

  • 1. x-data="{ image: 'image1' }" to declare a new Alphine component and its data for a block of HTML.
  • 2. x-show="image === 'image1'" to Toggle the visibility of an element (First Large Image).
  • 3. x-show="image === 'image2'" to Toggle the visibility of an element (Second Large Image).
  • 4. x-show="image === 'image3'" to Toggle the visibility of an element (Third Large Image).
  • 5. x-on:click.prevent="image = 'image1'" OR @click.prevent="image = 'image1'" to listen for browser events on an element (First Thumbnail Image).
  • 6. x-on:click.prevent="image = 'image2'" OR @click.prevent="image = 'image2'" to listen for browser events on an element (Second Thumbnail Image).
  • 7. x-on:click.prevent="image = 'image3'" OR @click.prevent="image = 'image3'" to listen for browser events on an element (Third Thumbnail Image).
  • 8. x-bind:class="{ 'border-blue-500' : image === 'image1'}" OR :class="{ 'border-blue-500' : image === 'image1'}" to dynamically set HTML attributes on an element.
  • 9. x-bind:class="{ 'border-blue-500' : image === 'image2'}" OR :class="{ 'border-blue-500' : image === 'image2'}" to dynamically set HTML attributes on an element.
  • 10. x-bind:class="{ 'border-blue-500' : image === 'image3'}" OR :class="{ 'border-blue-500' : image === 'image3'}" to dynamically set HTML attributes on an element.
