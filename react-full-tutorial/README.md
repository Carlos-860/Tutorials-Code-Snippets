# Project Setup

1. Run the following command to create a directory for your react app. Replace **my-app** with the applications name.

```Shell
npx create-react-app my-app
```

2. Change directory into the react application.

```Shell
cd my-app
```

3. The following commands starts the development server

```Shell
npm start
```

## Conditional rendering

- Use the style attribute with ternary operators.
- Use the && to only display truthy values on the right of it.

## Props

- key pair values`title={object name.property}`
- whole object `item={<object name>}`
- spread object `{...<object name>}`
